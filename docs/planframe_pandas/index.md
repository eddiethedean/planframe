# planframe-pandas

This track is for **end users** of the pandas adapter (`planframe_pandas`).

`PandasFrame` subclasses the core **`planframe.pandas.PandasLikeFrame`** mixin, so pandas-style helpers are available alongside the shared PlanFrame `Frame` API. For the full list of behaviors and typing notes, see the core guide: [Pandas-like API (`planframe.pandas`)](../planframe/guides/pandas-like-api.md).
